# 🚀 Fake Store API Test Framework

Automated API test framework built with Java, RestAssured, Maven, Allure and CI/CD.

---

## 📌 About Project

This project is a structured API automation framework for testing the Fake Store API:

🔗 https://fakestoreapi.com/

It includes:

- REST API automation with RestAssured
- Clean architecture (Services / Models / Core)
- SOLID principles
- Allure reporting
- CI/CD ready setup (GitHub Actions)
- Smoke test structure

---

## 📊 Allure Reporting

This framework is integrated with Allure for detailed reporting:

- Request & Response logs
- Test steps
- Status tracking
- History

---

## 🔄 CI/CD

The project supports CI/CD via GitHub Actions.

Pipeline includes:

1. Build
2. Run tests
3. Generate Allure results
4. Store artifacts
